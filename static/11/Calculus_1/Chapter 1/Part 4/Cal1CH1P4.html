فعالیت

می‌خواهیم نمودار تابع $y = \big| \ x - 1 \ \big| + 2$ را رسم کنیم.

روش اول: با توجه به نمودار $y = \big| \ x \ \big|$ در شکل (۱) و استفاده از انتقال‌های مناسب، نمودار آن را رسم کنید.

روش دوم: گام اول: با استفاده از تعیین علامت، تابع را به صورت ترکیبی از دو ضابطه‌ای بنویسید.

$$ y = \big| \ x - 1 \ \big| + 2 = \begin{cases} \qquad , & x \geq 1 \\ \\ \qquad , & x < 1 \\\\ \end{cases} $$

ویژگی‌های قدر مطلق


در سال‌های قبل با برخی از ویژگی‌های قدر مطلق آشنا شده‌اید که عبارت‌اند از:


الف) $ \big| \ x \ \big| \geq 0$

ب) $ \big| \ x \ \big| = a \iff x = a \, \ \text{یا} \ \, x = -a \qquad (a \geq 0)$

ت) $ \big| \ -x \ \big| = \big| \ x \ \big| $

ب) $\sqrt{x^2} = \big| \ x \ \big|$

ت) $ \big| \ x \ \big| = \big| \ a \ \big| \iff x = a \, \ \text{یا} \ \, x = -a$

ج) $ \big| \ x \ \big| ^2 = x^2$



فعّالیت

فرض کنید $a$ و $b$ اعدادهای حقیقی دلخواه باشند.

۱) از رابطه $\sqrt{a^2} = \big| \ a \ \big|$ استفاده کنید و نشان دهید که: $$ \big| \ ab \ \big| = \big| \ a \ \big| \ \big| \ b \ \big|$$

۲) با فرض $$b \neq 0$$ و استفاده از مرحله قبل ثابت کنید که:

$$\left| \frac{a}{b} \right| = \frac{ \big| \ a \ \big| }{ \big| \ b \ \big| }$$



فعّالیت

۱) فرض کنید $c$ یک عدد حقیقی نامنفی باشد. هر یک از نمادها‌های زیر را به جواب متناظر آن وصل کنید.


الف) $ \big| \ x \ \big| < c, \, (c \neq 0)$ (۱)

ب) $ \big| \ x \ \big| > c$ (۲)

پ) $ \big| \ x \ \big| \leq c$ (۳)

ت) $ \big| \ x \big| \geq c$ (۴)



۲) برای هر عدد حقیقی $a$ نشان دهید که: $$- \ \big| \ a \ \big| \ \leq a \leq \ \big| \ a \ \big|$$


۳) برای هر دو عدد حقیقی $a$ و $b$ ثابت کنید که: $$ - \ \big| a \ \big| \ - \ \big| \ b \ \big| \leq \ a + b \ \ \leq \ \big| \ a \ \big| \ + \ \big| \ b \ \big| $$


۴) با استفاده از قسمت قبل «نامساوی مثلث» را برای هر دو عدد حقیقی $a$ و $b$ نتیجه بگیرید:

$$\big| \ a + b \ \big| \ \leq \ \big| \ a \ \big| \ + \ \big| \ b \ \big|$$


معادلات قدر مطلق


حل یک مسئله

بر روی محور اعداد حقیقی فاصله چه نقاطی از نقطه ثابت $7$ برابر $3$ است؟

برای حل مسئله شکل روبه‌رو را رسم می‌کنیم:

اگر طول نقطه جواب مسئله را $x$ بنامیم، شرط مسئله به این معناست که: $ \big| \ x - 7 \ \big| = 3$ با استفاده از ویژگی‌های قدر مطلق خواهیم دانست که: $ x - 7 = \pm 3 $ و در نتیجه، $x = 4$ و $x = 10$ هر دو جواب‌های معادله هستند.



جواب‌های معادله $ \ \big| \ f(x) \ \big| \ = \ \big| \ g(x) \ \big| \ $ همان جواب‌های دو معادله $f(x) = - \ g(x)$ و $f(x) = g(x)$ هستند. به معادلاتی نظیر این معادلات که شامل عبارت قدر مطلق هستند معادلات قدر مطلقی می‌گویند.



مثال: معادله $ \ \big| \ x - 4 \ \big| \ = \ \big| \ 3x - 2 \ \big| \ $ را حل کنید.

روش اول: با استفاده از ویژگی‌های قدر مطلق: جواب‌های این معادله همان جواب‌های دو معادله $ 3x - 2 = x - 4$ و $x - 4 = - \ (3x - 2)$ هستند که، به ترتیب، عبارت‌اند از:

$$x = \dfrac{3}{2} \quad \text{و} \quad x = - \ 1$$


روش دوم: با به توان دو رساندن طرفین معادله خواهیم داشت: $$ 9x^2 - 12x + 4 = x^2 - 8x + 16$$ و از آنجا: $$ 2x^2 - x - 3 = 0 $$ جواب‌های این معادله $ \dfrac{3}{2}$ و $ - \ 1 $ هستند.
